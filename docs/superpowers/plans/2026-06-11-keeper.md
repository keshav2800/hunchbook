# Keeper Service Implementation Plan

> Executed inline 2026-06-11. Compact plan — design approved in-session.

**Goal:** `scripts/src/keeper.ts` — the vault's robot operator. Four isolated jobs per cycle,
signed with the CLI keystore key (reuses `lib/signer.ts`), terminal process for the demo.

**Verified pre-flight:**
- PLP price is exactly computable off-chain: Predict object fields `vault.balance − vault.total_mtm`
  ÷ `treasury_cap.total_supply` → 1.002220 vs 1.002232 actual execution (match; gap = pool drift).
- Pool utilization = `total_max_payout / vault_value` → 16 bps today (quiet → 1%/yr hedge budget
  per `hedge_policy::annual_hedge_bps`).
- `lib/signer.ts` loads the operator keypair from the sui CLI keystore; never prints secrets.
- `shared/indexer.ts` has `listOracles`, `getOracleState` (spot), `pickRoundTripOracle`.

**Jobs (each in its own try/catch — a hedge failure must never stop mark posting):**
1. **Mark** — target = vault_value×2⁶⁴/plp_supply; post `set_plp_mark` only when the change
   ≥ MARK_MIN_DELTA_BPS (default 2bps; saves gas, no event spam). Steps toward targets beyond
   the contract's ±20% drift bound in capped hops.
2. **Fees** — `accrue_fees` hourly (skipped while the vault has no shares).
3. **Buffer** — keep idle within [25%, 60%] of NAV, rebalancing toward 40%:
   low → `redeem_plp_to_idle`, high → `supply_idle_to_plp` (skipped while paused).
4. **Hedge** — (a) redeem expired *winning* wings (`redeem_hedge_wing`) and `reclaim_idle`
   the manager balance (capped at deployed_principal); (b) if no live wing: pick a BTC oracle
   10min–6h out, strike = spot × (1 − hedge_sigma_bps), DOWN wing (crash insurance), budget =
   max(NAV × annual_hedge_bps(util) × tte/year, HEDGE_MIN_BUDGET_USD floor — annualized math
   yields dust at $100 testnet scale), quantity from devInspect ask quote; `deploy_idle` +
   `mint_hedge_wing` in one atomic PTB.

**Ops hygiene:** heartbeat log line per cycle (NAV, share price, mark, util, idle%, gas);
gas-balance warning below 0.05 SUI; DRY_RUN env for no-write runs; SIGINT graceful exit;
env-tunable intervals/thresholds with sane defaults.

**Files:** create `scripts/src/keeper.ts`; add `"keeper": "tsx src/keeper.ts"` to scripts
package.json. No contract or web changes.

**Verify:** one DRY_RUN cycle (logs all four decisions), then one live cycle with
MARK_MIN_DELTA_BPS=0 to force a real `set_plp_mark` post; confirm PlpMarked event + vault page
NAV history gains a point.
